# Stitch MCP — Setup

Stitch MCP is configured and working in this project.

- **Config:** `.cursor/mcp.json` — stitch server runs via `.cursor/scripts/stitch-mcp-run.sh` (wrapper loads API key from `.cursor/stitch-api-key`).
- **Local install:** `@_davideast/stitch-mcp` is installed in `.cursor/` so MCP starts without running `npx`.

To re-run setup or change auth, see the official [davideast/stitch-mcp](https://github.com/davideast/stitch-mcp) repo.
